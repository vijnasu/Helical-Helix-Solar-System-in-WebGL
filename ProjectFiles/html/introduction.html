Introduction

The project “Helical Helix Solar System”, a representative view of Solar System in a new perspective.It mainly deals with rotation of planets around the Sun in Helical path. The sun is moving with certain speed and the planets follow identical helical orbits. The solid angle at the source of light and the distances to each planet are maintained uniformly.


WebGl is the tool used to implement the 3d view of new interpretation of Solar system .We have also used THREE.js java script file to make our implementation simple.
