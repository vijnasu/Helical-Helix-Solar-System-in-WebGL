Description

The Sun is moving with certain speed and the planets follow identical Helical orbits with different orbital velocities. The solid angle at the source of light and the distances to each planet are maintained uniformly. WebGl is the tool used to implement the 3D view of new interpretation of Solar system. An abstraction of WebGL called as THREE.js has been used for implementation. The fractal geometry required to represent the planet is animated using SphereGeometry from THREE.js. Texture mapping is done on the sphere created thereby. The dimensions of the sphere will be relatively proportional to the value proposed by NASA. The texture is obtained from the NASA website http://www.nasa.gov/multimedia/3d_resources/images.html. Then rotation transformation about primary axes has been applied to implement the revolution of the planets.

Planets rotate in Helical orbit as shown in figure:

The rotation is made possible by using the equation for x ,y ,z for the sphere that is added to Object3D. The equation used is
x(t)=Rcos(t)
y(t)=Rcos(t)
z(t)=t
where, R is the orbital radius.

By varying the value of parameter 't', the planetary rotation velocity is maintained.
